# Hero War 英雄戰爭 - VS2022 C++ GDI 遊戲範例

## 專案說明
這是一個使用 Win32 API 和 GDI 繪圖的 C++ 遊戲範例，設計用於練習 GitHub Copilot 功能。

## 遊戲特色
- 🦸 **向量英雄**：由多個幾何圖形組成的英雄角色
- 👹 **各級怪獸**：不同等級的怪獸散布地圖
- ⚔️ **武器選擇**：劍（快速攻擊）vs 斧頭（高傷害）
- 📊 **狀態顯示**：角色頭上顯示等級與血量
- 🎮 **鍵盤控制**：方向鍵移動，A鍵攻擊

## 操作方式
| 按鍵 | 功能 |
|-----|------|
| ↑ ↓ ← → | 移動英雄 |
| A | 攻擊附近怪獸 |
| 1 | 選擇劍（遊戲開始時） |
| 2 | 選擇斧頭（遊戲開始時） |
| ESC | 結束遊戲 |

## 建置方式
1. 使用 Visual Studio 2022 開啟 `HeroWar.sln`
2. 選擇 Debug/x64 配置
3. 按 F5 編譯並執行
